📚 Документация демоэкзамена 2026

В данном разделе находятся официальные материалы демонстрационного экзамена.

📄 Содержимое

1. Основной документ (модули 1 и 2)

📌 Включает:
- описание структуры экзамена
- модули 1 и 2
- пример задания
- критерии оценивания
- требования к оборудованию

2. Шаблон отчёта

📌 Используется во время экзамена:
- заполняется в процессе выполнения задания
- содержит рисунки, пояснения, таблицы
- является частью итоговой оценки

❗ Примечание

Все материалы размещены исключительно в образовательных целях.
