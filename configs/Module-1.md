# 📘 Модуль 1 — Пояснения к настройке сети 

## 1. Настройка имён устройств

### 🍃 Маршрутизаторы (HQ-RTR, BR-RTR)

Основные режимы конфигурирования EcoRouter:

ecorouter> - пользовательский режим. Режим по умолчанию при подключении. Доступны базовые команды просмотра состояния (например, show)

ecorouter#, команда enable - привилегированный режим. Позволяет просматривать детальную конфигурацию и выполнять диагностику.

ecorouter(config)#, команда configure terminal - глобальная конфигурация. Используется для изменения глобальных настроек
