# Различные вспомогательные файлы от меня и из других источников

## Описание

Здесь могут находиться:

* мои заметки
* черновики конфигураций
* дополнительные материалы
* файлы, используемые при подготовке

## P.S

Здесь также лежат разные тестовые и просто прикольные файлы, не относящиеся напрямую к проекту.

> [!NOTE]
> Файлы из этой директории не являются основной частью проекта и используются как вспомогательные.
