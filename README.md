# 🧪 Демонстрационный экзамен 2026

## Сетевое и системное администрирование (09.02.06)

---

## 📌 О проекте

Данный репозиторий посвящён созданию домашнего стенда и полной реализации демонстрационного экзамена 2026 года.

Проект включает:

* разбор заданий
* пошаговую реализацию модулей
* конфигурации
* практические примеры

📎 Репозиторий создан обычным студентом колледжа, заканчивающим обучение по специальности **сетевого и системного администратора**, с целью упростить подготовку к экзамену как для себя, так и для других.

---

## 🎯 Цель

Помочь студентам подготовиться к демонстрационному экзамену через практику и реальную настройку инфраструктуры.

---

## 🧰 Что будет в репозитории

* 🔧 Развёртывание лабораторного стенда
* 🌐 Настройка сетевой инфраструктуры (Модуль 1)
* 🖥 Организация сетевого администрирования (Модуль 2)
* 📦 Готовые конфигурации
* 📸 Скриншоты выполнения
* 📄 Примеры заполнения отчётов

---

## 🛠 Используемые инструменты

* VMware Workstation
* (в будущем возможно добавление других инструментов)

---

## 🚀 Статус проекта

🟡 В разработке

## 📂 Структура проекта

```
realization   - развёртывание стенда и способы реализации  
images-r      - скриншоты процесса развёртывания  
images-m1     - скриншоты выполнения модуля 1  
images-m2     - скриншоты выполнения модуля 2  
docs          - официальная документация экзамена  
configs       - конфигурации с пояснениями  
mydocs        - личные заметки, схемы, таблицы IP и доп. материалы  
```
